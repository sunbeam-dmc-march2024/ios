# Blogs Server

## installation

```bash

# install global utilities
> sudo npm install -g nodemon

# nodemon: used to restart the server automatically when the source code get updated

# install required packages
> npm install express mysql2 jsonwebtoken cors crypto-js morgan multer uuid

# express: for REST APIs
# mysql2: for mysql connectivity
# jsonwebtoken: JWT implementation
# cors: for cross origin resource sharing
# crypto-js: for password encryption
# morgan: for logging
# multer: for uploading files
# uuid: for generating a unique value (for image file)

```
